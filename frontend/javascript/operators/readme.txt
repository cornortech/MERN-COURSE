
/**

 * 
 * 1. Arithmetic Operators
 *  - Addition (+)
 * - Subtraction (-)
 * - Multiplication (*)
 * - Division (/)
 * - Modulus (%)
 * - Exponentiation (**)
 * - Increment (++)
 * - Decrement (--)
 * 
 *

 * - Assignment (=)

 * - Subtraction Assignment (-=)
 * - Multiplication Assignment (*=)
 * - Division Assignment (/=)
 * - Modulus Assignment (%=)
 * - Exponentiation Assignment (**=)
 * 

 * * 3. Comparison Operators (checks true of false )
 * - Equal (==)
 * - Strict Equal (===)
 * - Not Equal (!=)
 * - Strict Not Equal (!==)
 * - Greater Than (>)
 * - Less Than (<)
 * - Greater Than or Equal (>=)
 * - Less Than or Equal (<=)
 * 
 * 
 * * 4. Logical Operators
 * - Logical AND (&&)
 * - Logical OR (||)
 * - Logical NOT (!)
 *
